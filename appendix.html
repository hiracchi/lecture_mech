4. 付録¶

4.1. 用語集¶

HOMO(Highest Occupied Molecular Orbital)

最もエネルギーの高い占有軌道

LUMO(Lowest Un-occupied Molecular Orbital)

最もエネルギーの低い空軌道

4.2. 量子化学計算で使われる単位¶

` 1 a.u. = 627.5095 kcal mol-1 = 27.2114 eV `

4.3. WebMOの使い方¶

4.3.1. はじめに¶

WebMO (https://www.webmo.net/) はwebブラウザで分子モデリングを行い、GaussianやGAMESSなどの量子化学計算パッケージへのインターフェースを備えたサーバーサイドのプログラムである。

詳細は、本家のドキュメント を参照のこと。

4.3.2. 分子構造の作り方¶

  1. ブラウザからWebMOのURLを開き、ログインする。WebMOの "Job Manager" のページになる。

  2. 新しい分子を作成するには、 New Job ‣ Create New Job を選択する。"Build Molecule" のページに移る。

  3. Buildアイコン を選ぶと、現在選択している元素とヒントが下部のステータスバーに表示される。

  4. キャンバスのどこかで左クリックすると、原子が配置される。

  5. やり直したい場合は、Edit ‣ Undo を選択する。

  6. 原子種を変えたい婆は、Periodic Tableアイコン をクリックして原子種を変更する。

  7. 原子をクリックしてからドラッグすると、新しい原子が配置されるとともに、結合が形成される。

  8. 結合次数を変更する場合は、Adjustアイコン をクリックして adjust モードにしてから、結合を選択し、右クリックする。

  9. 原子の混成軌道(hybridization)を変更する場合も、Adjustアイコン をクリックして adjust モードにしてから、原子を選択し、右クリックする。

ちなみに

水素を自動的に付加したい場合は、Cleanup ‣ Add Hydrogens を選択する。ただし、混成軌道や結合次数を正しく設定していない場合は、失敗する。

ちなみに

構造を分子力学方を用いた構造最適化する場合は、Comprehensive Cleanup using Mechanicsアイコン をクリックする。

4.3.3. ジョブの設定と実行¶

  • キャンバス下部にある矢印 をクリックすると "Configure Gamess Job Options" に移る。

  • 計算エンジンの選択が表示されている場合は、 "Gaussian" を選択する。

  • 各タブをクリックして計算条件を設定する。計算条件を設定した後、下部にある矢印 をクリックすると計算が "Job Manager" に登録される。

注意

ジョブの実行に際し、ローカルルールがある場合はそれに従うこと。

4.3.3.1. Job Options¶

  • Job Name には任意の名前を記入します。"Job Manager" にはこの名前が表示される。

  • Calculation は計算目的とする物性を選択する。

  • Theory では理論的手法を選択する。

  • Basis Set では基底関数セットを選択する。

  • Charge は分子の電荷を入力する。

  • Multiplicity はスピン多重度を設定する。

4.3.3.2. Advanced¶

  • Save Checkpoint File をチェックするとCheckpointファイルを作成する。Checkpointファイルには途中計算結果などが含まれているため、再計算するときに便利。

  • Cartesian Coordinates をチェックすると原子座標をCartesian座標で入力する。

  • 分子軌道を表示したいときは、Additional Keywords に "Pop=Full GFInput"を入力する。

4.3.3.3. Preview¶

  • Generate をクリックすると、Gaussianの入力ファイルが表示される。

4.3.4. ジョブの管理 (Job Manager)¶

  • "Job Manager" には、実行中と完了したジョブがリストアップされる。

  • 完了したジョブは、右側の拡大鏡アイコンをクリックすると結果を見ることができる。

注意

http(s)の特性上、サーバー側から更新をpush通知することができず、webページの自動更新には限界がある。 Refresh をクリックしてジョブの状況を随時更新すること。

4.3.5. 結果の表示 (View Job)¶

  • 左側"Actions"にある Raw output をクリックすると、計算の全出力(生データ)が表示される。

  • 等値面(isosurface)の値は、 をクリックして、"Preferences" ダイアログから変更できる。

  • "Natural Population Analysis" では、 をクリックすると、電荷に応じて原子が色付けされる。

  • (7-11行目) 次に分子指定セクションを入力する。cartesianまたはz-matrixで入力する。仮に座標が整数値であっても浮動小数点型で計算させるように(計算精度確保のため)、小数点を入力した方が良い。 最後は空行で終える 。

原子座標の単位はオングストローム(Å)である。

注釈

4.4.1.2. スピン多重度¶

スピン多重度は以下の式により求められる。

\[スピン多重度=2S+1\]

ここで \(S\) は分子の全スピンをあらわし、 \(α=+1/2, β=-1/2\) である。

4.4.2. 出力ファイル¶

4.4.2.1. Mulliken電荷¶

Mullikenの電子数解析において、原子 \(A\) に属する電子数 \(Q_{A}\) は以下の式によって得られる。

\[Q_{A} = \sum_{p}\sum_{q \in A}{P_{pq} S_{pq}}\]

ここで \(\boldsymbol{P}\) は密度行列、 \({\boldsymbol{S}}\) は重なり積分をあらわす。 したがって原子 \(A\) のMulliken電荷 \({q_{A}}\) は、原子核の電荷 \(Z_{A}\) を用いて次のようにあらわされる。

\[q_{A} = Z_{A} - Q_{A}\]

Mulliken電荷は基底関数依存性があるものの、簡単に計算できるため、計算結果のチェックに便利である。
